# 全网资源导航插件

## 简单介绍

- 通过整理最好用的搜索引擎，方便您搜索需要的资源
	- 视频资源
	- 学习资料
	- 网盘资源
	- 更多
- 使用步骤
  1. 在网页按 Alt + J 跳转聚合导航页面
  2. 选择搜索引擎
  3. 输入搜索关键词搜索
- github传送门：[https://github.com/Anjude/tampermonkey](https://github.com/Anjude/tampermonkey)
